Getting Started

Welcome to Finishing Labs ERP! This guide will help you navigate and use all features of the system.

Logging In

Access the system through your web browser at the URL provided by your administrator. Use your assigned credentials to log in.

Navigation

The system uses a sidebar navigation menu to access different modules:

  • Dashboard: Overview of key metrics and recent activity
  • Customers: Manage customer information
  • Orders: Create and track customer orders
  • Packing Slips: Generate shipping documentation
  • Production: Manage production workflow

Dashboard

The dashboard provides a real-time overview of your business operations, including:

  • Total number of customers
  • Active orders
  • Recent activity feed
  • Production status summary

Customer Management

Viewing Customers

Navigate to the Customers section to view a list of all customers in the system. Use the search and filter options to find specific customers.

Adding a New Customer

  1. Click the "New Customer" button
  2. Fill in the customer information form:
    • Company name
    • Contact person
    • Email and phone
    • Billing and shipping addresses
  3. Click "Save" to create the customer record

Editing Customer Information

Click on any customer in the list to view their details. Click "Edit" to update their information.

Order Management

Creating an Order

  1. Navigate to Orders and click "New Order"
  2. Select the customer from the dropdown
  3. Add order details:
    • Order date
    • Due date
    • Items and quantities
    • Special instructions
  4. Review and submit the order

Tracking Orders

View all orders in the Orders list. Orders are color-coded by status:

  • Pending: Awaiting processing
  • In Production: Currently being manufactured
  • Ready to Ship: Completed and awaiting shipment
  • Shipped: Dispatched to customer
  • Completed: Delivered and closed

Packing Slips

Generating a Packing Slip

  1. Navigate to Packing Slips
  2. Click "Create Packing Slip"
  3. Select the order
  4. Review the automatically populated information
  5. Click "Generate" to create the packing slip
  6. Print or download the PDF

Production Management

Production Workflow

The Production module helps track items through the manufacturing process.

Production Builder

Use the Production Builder to:

  • View orders ready for production
  • Assign items to production queues
  • Update production status
  • Mark items as completed

Tracking Production Status

Monitor the status of all items in production from the Production overview page. Filter by status, customer, or date range.

Tips & Best Practices

  • Always verify customer information before creating orders
  • Update order status promptly to maintain accurate tracking
  • Generate packing slips before shipping to ensure accuracy
  • Use the dashboard regularly to monitor overall system health
  • Keep notes in order comments for future reference
