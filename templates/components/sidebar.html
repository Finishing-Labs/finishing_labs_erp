{# SIDEBAR NAVIGATION COMPONENT HOW IT WORKS: - Fixed position on left side of screen - Uses HTMX for AJAX navigation without page reloads - When user clicks a link, HTMX: 1. Makes GET request to Flask route (hx-get) 2. Swaps response into #content div (hx-target) 3. Updates browser URL (hx-push-url) 4. No full page reload - just the content area changes! HTMX ATTRIBUTES EXPLAINED: - hx-get="/route" → Makes AJAX GET request to this URL - hx-target="#content" → Puts response HTML into element with id="content" - hx-swap="innerHTML" → Replace inner HTML (default behavior) - hx-push-url="true" → Update browser URL bar for bookmarking/back button ACTIVE STATE: - Uses Jinja2 conditional to highlight current page - request.endpoint contains Flask route (e.g., 'dashboard.index') #}
